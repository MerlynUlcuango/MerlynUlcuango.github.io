SSL

Secure Sockets Layer (Capa de sockets seguros, en español). Es un protocolo criptográfico que proporciona comunicación segura a través de una red, generalmente internet. SSL garantiza que los datos transmitidos entre un servidor y un cliente (como un navegador web) estén encriptados y se mantengan confidenciales.

Burp Suite

Burp Suite es una herramienta de seguridad informática utilizada para realizar pruebas de seguridad en aplicaciones web. Es ampliamente utilizada por profesionales de la seguridad, pentesters y expertos en seguridad de aplicaciones para identificar vulnerabilidades y debilidades en los sistemas web.

Dominio

Se refiere a una identificación única y legible por humanos que se utiliza para identificar y acceder a recursos en la web, como sitios web, servidores de correo electrónico y servicios en línea. Es una parte fundamental de las direcciones web y se utiliza para localizar y distinguir diferentes entidades en internet.
